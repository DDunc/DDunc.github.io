Clade: JavaScript and Regex

Clade is a work in progress with the eventual goal of being able to recreate the historical vowel and consonant shifts between English and German orthography. Because English and German are part of the same West Germanic language family, this tool could help language learners identify previously unrecognized cognates and rapidly expand their active vocabulary.

Clade Github

Sunk: JavaScript and jQuery

Sunk is a small game conceived of as a hybrid of interactive supernatural fiction and resource management. This was a learning project for jQuery, CSS, Bootstrap, and becoming familiar with native JavaScript methods, getters and setters, etc.

Gut Feeling: JavaScript, jQuery, Pixi

Gut Feeling is my first collaborative project and very much still a work in progress. The player takes the role of the stomach's gut flora and must manage various bacteria in order to optimize digestion and avoid health issues.
